# Workout Log - Мобильное приложение для трекинга тренировок

Современное мобильное приложение для отслеживания тренировок с минималистичным дизайном и интуитивным интерфейсом.

## Особенности

- 📱 Адаптивный дизайн для мобильных устройств
- 🎨 Минималистичный современный интерфейс
- 📊 Статистика тренировок
- 📝 История тренировок
- ➕ Удобное добавление новых тренировок
- 🏋️ Трекинг сетов (вес / повторения / подходы)

## Установка и запуск

### Требования

- Node.js (версия 14 или выше)
- npm или yarn

### Установка зависимостей

```bash
npm install
```

### Запуск приложения

```bash
npm start
```

Приложение откроется в браузере по адресу `http://localhost:3000`

### Сборка для продакшена

```bash
npm run build
```

## Использование

1. **Главный экран**: Отображает историю тренировок, список упражнений и статистику
2. **Добавление тренировки**: Нажмите на кнопку "+" в правом нижнем углу
3. **Создание сета**: В экране создания тренировки добавьте сет и укажите вес, повторения и количество подходов
4. **Сохранение**: Нажмите "Сохранить" для сохранения тренировки
